Instructions - (for ubuntu users)
	1. do "mkdir build" in pacman-killer directory
	2. go into build directory. (cd build)
	3. do "cmake .."
	4. do "make"
	5. do "./packman-killer"

	OR 
	you can run from qt-creator

Controls -
	1. a - move left
	2. d - move right
	3. space - jump
	4. scroll - zoom in/out and arrow keys for panning

Note - Different colored balls have different points.
